# קשרים (Connections Game)

משחק "קשרים" בעברית, בהשראת משחק ה-Connections של הניו יורק טיימס.

## אודות המשחק
המטרה היא למצוא 4 קבוצות של 4 מילים שיש ביניהן קשר משותף. המילים מוצגות בלוח של 16 משבצות.

## תכונות
- **טעינה דינמית**: המשחק טוען קטגוריות באופן אקראי מתוך קובץ `word.json`.
- **רמות קושי**: בכל משחק נבחרות 4 קטגוריות בדרגות קושי שונות (1 עד 4), המיוצגות על ידי צבעים:
  - 🟨 **צהוב**: קטגוריה קלה
  - 🟩 **ירוק**: קטגוריה בינונית
  - 🟦 **כחול**: קטגוריה מאתגרת
  - 🟪 **סגול**: קטגוריה קשה (או קשרים מתוחכמים)
- **מערכת טעויות**: לרשותכם 4 פסילות לפני סיום המשחק.
- **עיצוב רספונסיבי**: מותאם לעבודה בדפדפנים ובמובייל.

## איך מפעילים
מכיוון שהמשחק משתמש ב-`fetch` כדי לטעון את קובץ ה-JSON, יש להריץ אותו באמצעות שרת מקומי (Local Server) ולא ישירות מפתיחת הקובץ בדפדפן.

ניתן להשתמש בכל שרת מקומי, לדוגמה:
```bash
# באמצעות Python
python3 -m http.server

# באמצעות Node.js (npx)
npx serve .
```
לאחר מכן פתחו את הכתובת `http://localhost:8000` (או הכתובת שהשרת מספק) בדפדפן.

## הוספת קטגוריות חדשות
ניתן להוסיף קטגוריות נוספות בקובץ `word.json` לפי המבנה הבא:
```json
{
  "category": "שם הקטגוריה",
  "words": ["מילה1", "מילה2", "מילה3", "מילה4"],
  "difficulty": 1
}
```
ניתן להוסיף כמה קטגוריות שרוצים לכל רמת קושי. בכל טעינה המשחק יגריל אחת מכל רמה.
